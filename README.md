# Urbanik – Sinergia: inteligencia predictiva del territorio

Proyecto desarrollado para el concurso **Datos al Ecosistema 2025**, enfocado en el análisis y visualización de hechos de seguridad y convivencia en el departamento de Santander, Colombia.  
A partir de datos abiertos oficiales, se construye un modelo analítico y una interfaz web que permiten identificar patrones y niveles de riesgo por municipio, apoyando la toma de decisiones de entidades territoriales y ciudadanía.

## 2. Descripción general del proyecto

**Santander Urbanik** integra un pipeline de datos, modelos de *machine learning* y una interfaz web para:

- Consumir datos abiertos relacionados con seguridad y convivencia en el departamento de Santander.
- Limpiar, unificar y transformar la información para su análisis.
- Aplicar modelos de analítica predictiva y descriptiva para estimar niveles de riesgo.
- Mostrar los resultados en un **mapa de calor interactivo**, con filtros por municipio, año y tipo de hecho.
- Facilitar la lectura de tendencias territoriales y apoyar la formulación de estrategias de prevención.
